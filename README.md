## Frontend Startup template for development
This is a startup template for frontend development.
Any one can use this template for design any kind of projects.



#### Features
- Boostrap 5
- Jquery
- Owl Carousel
- isotop
- mix it up
- wow js
- node sass
- font awesome
- themify icon

### Use System
-  Clone this repo
- install npm

'''php
npm install
'''
